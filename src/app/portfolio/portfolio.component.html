Alex Swift Emergency Contact



The Cobra or Alex Swift Emergency Contact Service is a software that provides the ability to contact your emergency contacts when in need. The system can be linked to a call centre where by if you are lost over a period a call can put forward to a list of your immediate contacts. The system is built using .NET Framework – C#, ASP.NET MVC, HTML, CSS, AngularJS, Bootstrap.
